¿Qué concentraciones utilizas de manera habitual?
  • Eau Fraîche (EF): 1-3%, duración corta, ideal para climas cálidos.
  • Eau de Cologne (EDC): 3-5%, duración moderada, ideal para el uso diario.
  • Eau de Toilette (EDT): 4-8%, duración media, versátil para diferentes ocasiones.
  • Eau de Parfum (EDP): 8-15%, duración prolongada, para la noche o eventos.
  • Parfum (B) o Extrait de Parfum: 15-30%, duración muy prolongada
